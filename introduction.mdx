# What is Data Command?

Data Command is a **data-driven command palette** for React, built on top of [shadcn/ui](https://ui.shadcn.com). It lets you load items from APIs, handle nested selections, and build anything from simple dropdowns to complex command menus.

Unlike traditional command palettes with static items, Data Command is designed for **dynamic, asynchronous data** - perfect for search interfaces, navigation menus, and interactive command bars.

## Quick example

Here's a simple Data Command with nested items:

```tsx
import { DataCommand } from "@/components/data-command";
import { User, SettingsIcon, BanknoteIcon } from "lucide-react";

function Example() {
  return (
    <DataCommand
      items={[
        {
          icon: <User />,
          label: "Profile",
          value: "profile",
          loadItems: async () => [
            {
              label: "Settings",
              value: "settings",
              icon: <SettingsIcon />,
              onSelect: () => alert("You have selected: profile/settings"),
            },
          ],
        },
        {
          icon: <BanknoteIcon />,
          label: "Billing",
          value: "billing",
          onSelect: () => alert("You have selected: billing"),
        },
      ]}
    />
  );
}
```
